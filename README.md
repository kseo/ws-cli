ws-cli
======
WebSocket Command Line Client written in Go

## Installation

```
go get github.com/kseo/ws-cli
```

## Usage

```
$ ws-cli -url ws://echo.websocket.org
connected (press CTRL+C to quit)
> hi there
< hi there
> are you a happy parrot?
< are you a happy parrot?
```
